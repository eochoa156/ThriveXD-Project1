Game Rules

Tic Tac Toe:

  • Player starts as X while the computer plays as Os
  • First to get 3 in a row diagonally, horizontally, or vertically wins
  • Be warned, the computer is a tough opponent!


Simon:

  • First, the computer goes, then the player
  • The player must press the corresponding button that matches the one the computer pressed
  • Try to score as high as you can!
